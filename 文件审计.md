# 文件审计

---

## 一、评测审计

以【**MII 题目导出.pdf**】为准，对比另一个文件中的题目及答案。

### 审计要求

1. **不按题目序号进行对比**，需根据题目内容在另一个文件中查找对应题目，相同题目的答案必须一致。
2. 答案只能为以下三种：
   - 不适用
   - 是
   - 否
3. 如存在以下情况，需**单独列出**：
   - 另一个文件中缺少对应题目
   - 另一个文件中题目没有填写答案
   - 两个文件中相同题目的答案不一致
   - 答案不是"不适用 / 是 / 否"三种之一
4. 所有答案为**"是"**的题目，**必须有对应截图作为佐证**。
5. 对缺少截图、截图不对应或无法证明结果的题目，需列入**问题清单**。
6. 题目及问题内容**不能被图片遮挡**，如存在遮挡导致无法核验题目、问题或答案内容的情况，需列入**问题清单**。

### 输出内容

1. 缺失题目清单
2. 缺失答案清单
3. 答案不一致清单
4. 答案格式异常清单
5. "是"项缺少截图或截图不符合要求清单
6. 题目或问题内容被图片遮挡清单

## 二、定级审计

所有审计内容均以以下目录中的三个**标准答案文档**为准：

> `/Users/shaoshuai/company/文档审计`

### 审计要求

1. 定级材料中的**所有内容必须与三个标准答案文档保持一致**。
2. **安全等级 k 的计算结果需重新计算核验**。
3. **I、R、V** 三个取值必须符合三个标准答案文档中的要求。
4. **不得仅依据提交文件内容判断**，所有结论均需回到三个标准答案文档中进行核对。
5. 如上传照片作为核对依据，需将文档中的相关内容与照片内容逐项对比；如存在内容不一致、缺失、表述不符或计算结果不一致的情况，需**单独列出**。
6. 如发现以下情况，需**单独列出**：
   - 内容与标准答案不一致
   - I、R、V 取值不符合标准答案
   - 安全等级 k 计算错误
   - 计算过程缺失或不完整
   - 定级结论与标准答案不一致
   - 依据引用错误或缺失
   - 文档内容与上传照片不一致

### 历史问题复核

目录下的【**历史审计记录.xlsx**】为以往提交的反馈记录，审计时需参考该文件：

1. 查看历史提交过的问题
2. 对照本次材料复核历史问题是否仍然存在
3. 如历史问题仍存在，**继续列入本次问题清单**
4. 如历史问题已整改，可在备注中说明已整改

### 输出内容

1. 定级内容不一致问题清单
2. I、R、V 取值问题清单
3. 安全等级 k 计算问题清单
4. 定级结论问题清单
5. 历史问题复核结果
6. 文档内容与上传照片不一致问题清单
